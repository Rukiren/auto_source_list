# auto_source_list
這是一支用來幫忙快速將 linux source.list 切換成 mirror.twds.com.tw 的腳本

## 使用方式
```
# 自動偵測
sudo ./set_mirror_twds.sh

# 指定為 debian 並輸入 bookworm
sudo ./set_mirror_twds.sh debian

# 指定為 kali
sudo ./set_mirror_twds.sh kali
```
